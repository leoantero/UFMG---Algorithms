# Algorithms I 

This repository contains projects, exercises, and algorithm implementations developed during the university course **Algorithms I (DCC206)**.

## Course topics
- Algorithm analysis and asymptotic complexity
- Proof of correctness
- Graph algorithms:
  - BFS and DFS
  - Connected components
  - Bipartite graphs
  - Topological sorting
  - Shortest paths
  - Minimum spanning trees
  - Maximum flow
- Algorithmic paradigms:
  - Induction and recursion
  - Divide and conquer
  - Greedy algorithms
  - Dynamic programming
- Computational complexity:
  - Polynomial reductions
  - NP-completeness
  - P vs NP

## Purpose
This repository is intended for academic and study purposes, serving as a record of my work and learning throughout the course.
